Board Game Exploration

My group created a 3-4 player board game with the main objective being getting to the north star. In order to acomplish the main objective you have to overcome obstacles placed by other players and hope that you'll roll the right number.

Initial Rules

  • 3-4 players
  • Each player must start at point labeled "start"
  • If a player rolls a 1, they are allowed to place an obstacle in front of an opponent player
